| **Description des missions**           | Dans le cadre du projet ANR ALMACIR, le/la stagiaire participera au développement d’une base de données web et de son écosystème en données libres et ouvertes (Linked Open Data – LOD), rassemblant les informations numismatiques sur les monnaies de l’Occident musulman médiéval (al-Andalus et ses marges, VIIIᵉ–XIIIᵉ siècle). La BDD repose sur la pile logicielle open-source Numishare, fondée sur le format XML et sur l’ontologie Nomisma.org, garantissant l’interopérabilité des données numismatiques dans le cadre du Web sémantique. **Missions du/de la stagiaire :**  participation au développement, à la structuration et à la documentation du dépôt GitHub du projet (wiki, site Hugo, scripts, etc.) ;  intégration et interopérabilité de Numishare avec d’autres outils développés au sein de l’IRAMAT (applications Python/Dash, base de données des analyses physico-chimiques, serveur IIIF, etc.) ; contribution à la chaîne de production, de transformation et de diffusion des données XML/RDF liées au projet ;  participation éventuelle à la valorisation scientifique via des pages web ou des démonstrateurs. **Compétences requises :** bonne maîtrise des technologies XML : XML / XSLT / XQuery ;  compréhension solide des technologies du Web sémantique : RDF / SPARQL ;  aisance avec les environnements et outils du web : serveur Linux, bash, SSH, Apache ;  capacité à travailler dans un environnement open-source et collaboratif. **Compétences appréciées (atouts) :** connaissance de Python, notamment pour des scripts de transformation ou d’API (Dash) ;  pratique de Git/GitHub (issues, pull requests, documentation) ;  intérêt pour la numérisation du patrimoine, les humanités numériques, ou la numismatique (non obligatoire). |
